Hardware Specifications
=======================

The project uses the Seeed Studio XIAO ESP32S3 module and several sensors.

XIAO ESP32S3
------------

- **Module**: Seeed Studio XIAO ESP32S3.
- **Processor**: ESP32-S3 Dual-core Xtensa LX7.
- **Wireless**: Wi-Fi and Bluetooth LE.

Sensors
-------

- **RM3100**: PNI RM3100cb 3-axis magnetometer via I2C.
- **MCP9808**: Microchip MCP9808 high-accuracy temperature sensor via I2C.
- **L76K**: Seeed Studio L76K GNSS module via UART.
- **SX1262**: Seeed Studio Wio-SX1262 LoRa module via SPI.

Pinout
------

Refer to the official Seeed Studio documentation for the XIAO ESP32S3 pinout.
